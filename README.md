# OpenGameEngine
Game engine maded on OpenGL, Python 3.12

## Why was this engine created?
First of all, like any **game engine**, it is designed to simplify the process of creating games.

## What kind of engine is this?
This **engine**, like some others, is provided as a library without a graphical user interface, which can complicate the process but gives you full control over your game. There are also downsides to this **game engine**, namely that it is written entirely in Python, which reduces its performance in complex games.

## Game Engine Features:
+ **OpenGL** + **glfw** - Lightweight cross-platform support and ease of use;
+ **Python** - Write game logic in pure Python, no visual editors;
+ **Arch** - Its small architecture makes it easy to explore the engine and extend it;
+ **Render** - It only supports 2D rendering and includes a batching system for drawing multiple objects.
